The syntax for generic functions:

```ts
function functionName<T>(param: T): T {
	return param
}
```

The `<T>` declares a **type parameter**. When you call the function, TypeScript
infers (or you specify) what `T` is:

```ts
identity<string>('hello') // Explicit: T is string
identity(42) // Inferred: T is number
```
