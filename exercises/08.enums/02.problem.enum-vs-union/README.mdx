# Enum vs Union Types

different tradeoffs.

```ts
// Enum approach
enum Color {
	Red = 'red',
	Green = 'green',
	Blue = 'blue',
}
function paintWithEnum(color: Color) {}
paintWithEnum(Color.Red) // Must use enum value

// Union type approach (inline)
function paintWithUnion(color: 'red' | 'green' | 'blue') {}
paintWithUnion('red') // Can use string directly
```
