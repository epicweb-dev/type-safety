# Discriminated Unions

ğŸ‘¨â€ğŸ’¼ Discriminated unions are a pattern where each type in a union has a common
"discriminant" property with a literal type value.

```ts
type Circle = { kind: 'circle'; radius: number }
type Rectangle = { kind: 'rectangle'; width: number; height: number }
type Shape = Circle | Rectangle
```

The `kind` property lets TypeScript narrow the type automatically.

```ts
function handle(shape: Shape) {
	switch (shape.kind) {
		case 'circle':
			return Math.PI * shape.radius ** 2
		case 'rectangle':
			return shape.width * shape.height
		default: {
			const _exhaustive: never = shape
			return _exhaustive
		}
	}
}
```
